📦 INSTALLATION (3 Steps)
═══════════════════════════════════════════════════════════

1. Install everything:
   cd ~/Documents/Private/ai-router
   ./scripts/bootstrap.sh
   
2. Install global command:
   npm run install-command
   
3. Done! Now use from anywhere:
   kochava --help

🚀 BASIC USAGE
═══════════════════════════════════════════════════════════

# Quick query (Claude-like)
kochava "format this: function foo(){return 1}"

# Interactive mode (like Claude --chat)
kochava --chat
kochava -c

# Show statistics
kochava --stats
kochava -s

# With file context
kochava --file mycode.ts "explain this"

# Help
kochava --help
